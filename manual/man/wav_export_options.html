WAV Export Options

From Audacity Development Manual
Jump to: navigation, search
WAV is an uncompressed and lossless audio format PCM audio. The format is in Microsoft's Little-Endian byte order.
WAV files are limited to a maximum size of 4GB - see this page for details of how that relates to maximum times.
Accessed by: File > Export > Export as WAV

Format Options

The only option with WAV export is the Encoding option.

The default Encoding is 16-bit PCM which you can change from the Encoding dropdown menu.

If you do change it Audacity will remember and re-use your previous setting on next use.
